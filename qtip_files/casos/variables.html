Variables

Son los valores de orientación de los sistemas fotovoltaicos.

El ángulo de inclinación es el que se forma entre el plano horizontal y la altura a la que se fija el panel. 0 grados significa que el panel está dispuesto totalmente en forma vertical, como una ventana y 90 grados que está dispuesto en forma horizontal como la superficie de una mesa (recomendado 32).

El acimut o ángulo acimutal indica hacia dónde está viendo el panel.

0 grados significa que está orientado hacia el norte, 90 grados orientado hacia el este, 180 orientado al sur (es lo más recomendable para México por estar en el hemisferio norte) y 270 orientado hacie el oeste.
